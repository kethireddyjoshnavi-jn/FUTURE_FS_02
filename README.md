# 📊 Client Lead Management System (Mini CRM)

The Client Lead Management System (Mini CRM) is a web-based application developed to manage customer leads efficiently. The system allows users to add, edit, delete, search, and manage client information while tracking the status of each lead.

This project demonstrates CRUD operations, database connectivity, backend integration, and responsive web application development.

---

## 🎯 Objectives

- Manage customer leads efficiently.
- Store client information securely.
- Perform CRUD operations.
- Learn backend development.
- Integrate frontend with database.
